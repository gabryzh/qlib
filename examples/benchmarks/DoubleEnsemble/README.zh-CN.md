# DoubleEnsemble
* DoubleEnsemble是一个集成框架，利用基于学习轨迹的样本重加权和基于洗牌的特征选择，来解决低信噪比和特征数量增加的问题。他们根据每个样本的训练动态识别关键样本，并通过洗牌的方式根据每个特征的消融影响引出关键特征。该模型适用于广泛的基础模型，能够提取复杂的模式，同时缓解金融市场预测的过拟合和不稳定性问题。
* Qlib中使用的此代码由我们自己实现。
* 论文：DoubleEnsemble：一种基于样本重加权和特征选择的金融数据分析新集成方法[https://arxiv.org/pdf/2010.01265.pdf](https://arxiv.org/pdf/2010.01265.pdf)。
